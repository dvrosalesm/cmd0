Get Started

Install in seconds

# Clone the repository
git clone https://github.com/dvrosalesm/cmd0.git

# Run the installer
cd cmd0
./install.sh
1

Installs dependencies via npm

2

Builds TypeScript and compiles the app

3

Registers global hotkey shortcut

4

Launches in your menubar

Commands

Quick reference

/0 Modify my source code. The agent reads, writes, and reloads itself.
/cancel Stop the current agent operation immediately.
/tasks Manage background tasks. Add recurring jobs or one-off reminders.
/snap Save a complete snapshot of the current state for later restoration.
/restore Restore from a previously saved snapshot.
/safe Restart in safe mode with restricted permissions and base files.
