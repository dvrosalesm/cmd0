Features

🔍 Web Search

Search DuckDuckGo and fetch any URL. Research without leaving your flow.

🌐 Browser Control

Automate Chrome via Playwright. Navigate, click, fill forms, take screenshots.

💻 System Access

Run bash commands, read/write files, send desktop notifications.

🖼️ Screenshots

Capture any screen region. Paste or attach images to your prompts.

🔄 Self-Modification

cmd0 can edit its own source code. Use /0 to evolve the agent.

⏰ Task Daemon

Schedule recurring or one-time tasks. Background runner checks every minute.

💾 Snapshots

Save and restore complete state. Experiment freely, roll back anytime.

🔐 Safe Mode

Start with --safe to restore base files and disable self-modification.

How it works

$ Press ⌘0 / Super+-
→ Floating chat appears
$ "Search for the latest AI news"
→ web_search: "latest AI news"
→ [results appear in chat]
$ /0 add a voice input button
→ anima_read, anima_write, anima_reload
→ [app restarts with new feature]

Install

# Clone and run the install script
git clone https://github.com/dvrosalesm/cmd0.git
cd cmd0
./install.sh
1
Installs dependencies via npm
2
Builds TypeScript and compiles the app
3
Registers shortcut — Cmd+0 (macOS) or Super+- (Linux/Hyprland)
4
Launches the agent in your menubar

Commands

/0

Modify own source code. The agent reads, writes, and reloads itself.

/cancel

Stop the current agent operation immediately.

/tasks

Manage background tasks. Add recurring jobs or one-off reminders.

/snap <name>

Save a complete snapshot of the current state.

/restore <name>

Restore from a previously saved snapshot.

/safe

Restart in safe mode with restricted permissions.

Configuration
